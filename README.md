Postogram
A Social Network Project Built with Django
Project Overview
Postogram is a social networking platform that allows users to create profiles, share posts with images and comments, search for friends, and engage with a news feed. The project is built using Django, a high-level Python web framework.

Features
User Authentication and Authorization: Users can register, log in, and manage their accounts.
User Profiles: Users can create profiles with avatars and information.
Posts: Users can create and share posts with images and comments.
User Search and Friendships: Users can search for friends and send friend requests.
News Feed: Users can view a news feed with posts from their friends.
Post Likes and Comments: Users can like and comment on posts.
User Settings and Profile Editing: Users can edit their profiles and settings.
Getting Started
To get started with Postogram, follow these steps:

Clone the repository: git clone https://github.com/nimblen/Postogram.git
Install the required dependencies: pip install -r requirements.txt
Create the migrations: python manage.py makemigrations
Run the migrations: python manage.py migrate
Start the development server: python manage.py runserver
